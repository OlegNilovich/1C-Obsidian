// 7. Точка слияния			- Объединяет паралельные процессы в один
// 8. Точка обработки		- Выполняем определенный алгоритм системой (код)

// 3. Построение карты маршрута
// 4. Визуализация хода бизнес-процесса
// 5. Проверка работы бизнес-процесса в режиме пользователя
// 6. Настройка адресации задач
// 7. Отображение задач по исполнителям. Форма “Мои задачи”
