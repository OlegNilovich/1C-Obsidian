Система компоновки данных позволяет не только выводить поля ресурсов в ячейках отчета, но и группировать по ним данные. Можно также указать выражение для расчета ресурса на языке выражений
системы компоновки данных и затем сгруппировать по этому ресурсу данные в отчете

### Отчет с группировкой по полю-ресурсу:

Выражение для расчета ресурса на основе вычисляемого поля **ОбъемПродаж**
```bsl
Выбор Когда Сумма(Стоимость) <= 3000 Тогда "Малый" Иначе
Выбор Когда Сумма(Стоимость) >= 10000 Тогда "Большой" Иначе "Средний" Конец Конец
```

##### Структура отчета:
Отчет содержит составную группировку по полям *ОбъемПродаж* и *Номенклатура* с вложенной
группировкой *Номенклатура*. В выбранных полях отчета указаны поля *Стоимость* и *ОбъемПродаж*
![[Pasted image 20240924045554.png]]

##### Результат:
В результате выполнения отчета в колонке Объем продаж выводится значение выражения.

Записи отчета группируются по значению ресурса *Объем продаж*:
- Сначала выводятся записи с *большим* (Сумма(Стоимость) >= 10000) объемом продаж, 
- Затем записи с *малым* (Сумма(Стоимость) <= 3000) объемом продаж 
- Затем записи со *средним* (3000 < Сумма(Стоимость) < 10000) объемом продаж
