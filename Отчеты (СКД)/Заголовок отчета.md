#ОТЧЕТЫ 
### Задача:
Требуется создать отчет по продажам с помощью **СКД** и создать произвольный заголовок с названием отчета и отображением периодов.

**Без заголовка:**
![[Pasted image 20240909160339.png]]

**С заголовком:**
![[Pasted image 20240909164219.png]]
### Решение:
##### Создание отчета:
1. Создаем отчет **Продажи_СозданиеЗаголовка**
2. Создаем и открываем **Схему компоновки данных**
3. Добавляем набор данных **Запрос** и открываем **Конструктор запроса**
![[Pasted image 20240909155057.png]]
4. Во вкладке **Таблицы и поля** выбираем нужны поля для отчета. Закрываем конструктор
![[Pasted image 20240909155307.png]]
5. Итоговый текст запроса:
```bsl
ВЫБРАТЬ
	ПродажиОбороты.Контрагент КАК					Контрагент,
	ПродажиОбороты.Номенклатура КАК				Номенклатура,
	ПродажиОбороты.КоличествоОборот КАК		Количество,
	ПродажиОбороты.СуммаОборот КАК				Сумма
ИЗ
	РегистрНакопления.Продажи.Обороты КАК		ПродажиОбороты
```
6. Возвращаемся к настройке **Схемы компоновки данных** и определяем **Ресурсы**
![[Pasted image 20240909155503.png]]
7. Параметры => Запрещаем **Незаполненные значения** и включаем **Использовать всегда**
![[Pasted image 20240909165221.png]]
8. Вкладка **Настройки** => Открываем **Конструктор настроек** => Выбираем **Список** => **Далее**
![[Pasted image 20240909155747.png]]
9. Выбираем поля для отображения в отчете, жмем **ОК**
![[Pasted image 20240909155917.png]]
10. Добавляем параметры **Начало периода** и **Конец периода** в пользовательские настройки
![[Pasted image 20240909160529.png]]
11. Отчет => Другие настройки => Выводить параметры => **Не выводить**
![[Pasted image 20240909160923.png]]
12. Отчет => Другие настройки => Макет оформления => **Античный**
![[Pasted image 20240909160143.png]]
##### Создание заголовка:
1. Создаем новую группировку **Детальные записи**
![[Pasted image 20240909161454.png]]
2. Устанавливаем имя для группировки **Заголовок**
![[Pasted image 20240909161637.png]]
3. Меняем вариант использования группировки: **Дополнительная информация**
![[Pasted image 20240909161828.png]]
4. Переносим группировку **Детальные записи** в подчинение группировке **Заголовок**
![[Pasted image 20240909163636.png]]
5. Переходим во вкладку **Макеты** и выбираем команду **Добавить макет группировки**
![[Pasted image 20240909162024.png]]
6. Указываем имя группировки: **Заголовок**, жмем **ОК**
![[Pasted image 20240909162121.png]]
7. Два раза кликаем по полю **Область** и жмем на кнопку выбора **[...]**
![[Pasted image 20240909162251.png]]
8. Выделяем произвольную группу ячеек, где будет располагаться заголовок, жмем **Enter**
![[Pasted image 20240909162505.png]]
9. Объединяем область заголовка в одну ячейку и выбираем параметр Заполнение: **Шаблон**
![[Pasted image 20240909162851.png]]
10. Заполняем параметры макета
![[Pasted image 20240909163017.png]]
11. В свойствах ячейки заголовка указываем формат даты **ДЛФ=DD**
![[Pasted image 20240909163934.png]]
